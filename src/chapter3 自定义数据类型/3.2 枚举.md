## **3.2.1 定义枚举**
枚举允许你通过列举所有可能的值来定义一个类型。与结构体不同，枚举是 “二选一” 或 “多选一” 的逻辑

枚举最简单的用法是定义一组离散的选项。定义如下所示：

```
fn main() {
enum IpAddrKind {
    V4,
    V6,
}
}
```
枚举的每个变体（Variant）可以拥有不同类型和数量的关联数据。

```
enum Message {
    Quit,                       // 没有关联数据
    Move { x: i32, y: i32 },    // 匿名结构体
    Write(String),              // 包含单个 String
    ChangeColor(i32, i32, i32), // 包含三个 i32
}
```
枚举可以像结构体一样使用 impl 来定义方法。如下所示：
```
fn main() {
enum Message {
    Quit,
    Move { x: i32, y: i32 },
    Write(String),
    ChangeColor(i32, i32, i32),
}

impl Message {
    fn call(&self) {
        // 在这里定义方法体
    }
}

let m = Message::Write(String::from("hello"));
m.call();
}
```

## **3.2.2 Option枚举**

Option 是标准库定义的另一个枚举。Option 类型应用广泛是因为它编码了一个非常普遍的场景，即一个值要么有值要么没值。

Rust 并没有空值，不过它确实拥有一个可以编码存在或不存在概念的枚举。这个枚举是 Option<T>，而且它定义于标准库中。

注意可以不需要 Option:: 前缀来直接使用 Some 和 None。使用如下所示:

```

fn main() {
let some_number = Some(5);
let some_string = Some("a string");

let absent_number: Option<i32> = None;
}
```
注意，如果使用 None 而不是 Some，需要告诉 Rust Option<T> 是什么类型的，因为编译器只通过 None 值无法推断出 Some 成员保存的值的类型。
