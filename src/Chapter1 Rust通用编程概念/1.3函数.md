## **1.3.1函数的定义**

Rust 中的函数定义以 fn 开始，后跟着函数名和一对圆括号。大括号告诉编译器函数体在哪里开始和结束。

可以使用函数名后跟圆括号来调用我们定义过的任意函数。

函数也可以被定义为拥有参数（parameter），参数是特殊变量，是函数签名的一部分。当函数拥有参数（形参）时，可以为这些参数提供具体的值（实参）。

在函数签名中，必须声明每个参数的类型，当一个函数有多个参数时，使用逗号分隔。函数的定义和调用如下所示：
```
fn main() {
    print_labeled_measurement(5, 'h');
}

fn print_labeled_measurement(value: i32, unit_label: char) {
    println!("The measurement is: {}{}", value, unit_label);
}
```

## **1.3.2语句和表达式**

Rust 是一门基于表达式（expression-based）的语言，函数体由一系列语句组成，也可选择以表达式结尾。

语句（statement）是执行一些操作但不返回值的指令，以;结尾。表达式（expression）计算并产生一个值，末尾没有分号。

例如let y = 6; 是一个语句，而6或者5 + 6，则都是一个表达式。具体示例如下所示：

```
fn main() {
    let y = {
        let x = 3;
        x + 1
    };

    println!("The value of y is: {}", y);
}
```

在这个例子中y的结果是 4。这个值作为 let 语句的一部分被绑定到 y 上。注意，x + 1 行的末尾没有分号。表达式的结尾没有分号。如果在表达式的末尾加上分号，那么它就转换为语句，而语句不会返回值。

## **1.3.3带有返回值的函数**

函数可以向调用它的代码返回值。我们并不对返回值命名，但要在箭头（->）后声明它的类型。

在 Rust 中，函数的返回值等同于函数体最后一个表达式的值。

使用 return 关键字和指定值，可以从函数中提前返回；但大部分函数隐式返回最后一个表达式。示例如下：

```
fn five() -> i32 {
    5
}

fn main() {
    let x = five();

    println!("The value of x is: {}", x);
}
```
